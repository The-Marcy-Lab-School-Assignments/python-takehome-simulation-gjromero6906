# NYC 311 Service Requests Analysis

## How to Run

1. Make sure you have Python 3 installed.
2. Navigate to this folder in your terminal.
3. Run the script:

python3 analysis.py

Output will be saved to `output.txt`. The console will confirm when the file has been written.

## What This Script Does

This script reads the `nyc_311_requests.csv` file and analyzes service request data. It counts how many requests are currently open, finds the most common complaint type, and totals requests per borough. The results are written to `output.txt` in the exact format required by the assignment.

This script uses only Python's built-in libraries: `csv`.
